Llanquihue Tour
Descripción

Sistema básico orientado a objetos para representar personas vinculadas a la agencia de turismo Llanquihue Tour.

Estructura del proyecto:

**Package app**
- Main.java                                                                                                                                                                         

**Package model**
- Persona.java
- Direccion.java
- Empleado.java

**Funcionalidades**
- Uso de herencia.
- Uso de composición.
- Encapsulamieno mediante atributos privados.
- Métodos getter y setter.
- Uso de toString().
- Creación de objetos desde Main.

**Ejecución**
- Abrit el proyecto en IntelliJ IDEA.
- Ejecutar la clase Main.java.
- Visualizar loos datos en consola.
